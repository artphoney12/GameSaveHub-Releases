# GameSave Hub

GameSave Hub is a desktop application for managing game save data in a local gaming environment.

It is designed for places such as gaming centers, LAN setups, and shared gaming PCs where multiple users may play on different machines.

The server manages users, game save locations, save data storage, and client access. Each player can use their own account so their game progress can be kept separate and available when they move to another PC.

The main purpose of GameSave Hub is to make game save handling easier, cleaner, and more consistent across multiple computers without requiring players to manually save or load their progress.
